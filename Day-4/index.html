HTML Table

HTML tables allow web developers to arrange data into rows and columns.

Table cells, rows & headers

  • Using td tag, create table cells, stands for table data
  • Using tr tag, create table rows, stands for table row
  • Using th tag, create table header, stands for table head, by defualt: th element are bold and centered

thead, tbody & thead elements

  • thead : Groups of header content in a table
  • tbody : Groups of body content in a table
  • tfoot : Groups of footer content in a table

HTML Table Borders

  • To add a border, use the CSS border property on table, th, and td elements.
  • To avoid having double borders, use border-collapse property to collapse.
  • To add color in each cell, use background-color property.
  • To make table borders rounds, use border-radius property.

HTML Lists

  • HTML lists allow web developers to group a set of related items in lists.

  • To organize content, create ordered and unordered lists using <ol> and <ul> tags. Each list item starts with the <li> tag.

    HTML Description Lists

    • HTML also supports description lists.
    • A description list is a list of terms, with a description of each term.
    • The <dl> tag defines the description list.
    • The <dt> tag defines the term(name).
    • The <dd> tag describes each term.

    HTML class & id Attributes

  • Classes (class) : Used to apply the same style to multiple elements. Used period(.) before class name and properties within curly-braces {}

  • IDs (id) : A unique identifier for a single element. use a hash chacter(#) before id name and properties within curly-braces{}.
